Available Calculation Tools

Braking Performance

Calculate braking performance parameters as per DIN EN 15746-2:2021-05 standard.

  • • Minimum braking force calculation
  • • Braking ratio analysis
  • • Deceleration computation

Hydraulic System

Design and analyze hydraulic brake systems with precision calculations.

  • • Cylinder force calculation
  • • Pressure analysis
  • • System efficiency

Load Distribution

Calculate axle load distribution and center of gravity for vehicles.

  • • Front/rear axle loads
  • • Dynamic load transfer
  • • CoG positioning

Q-max Analysis

Calculate maximum braking force and related parameters.

  • • Peak braking force
  • • Wheel lock analysis
  • • Adhesion limits

Tractive Effort

Analyze locomotive tractive effort and train resistance.

  • • Starting effort calculation
  • • Continuous effort
  • • Train resistance

Vehicle Performance

Complete performance analysis for vehicles.

  • • Acceleration analysis
  • • Speed calculations
  • • Power requirements

Key Features

Calculation History

All your calculations are automatically saved for future reference.

  • Give custom names to calculations (e.g., "EN-20 Project")
  • Load previous calculations with one click
  • Search through calculation history

Export Reports

Generate professional calculation reports for documentation.

  • Export to document formats
  • Professional formatting with all parameters
  • Include all input and output values

Dark Mode

Switch between light and dark themes for comfortable viewing.

Click the moon/sun icon in the navigation bar to toggle between themes. Your preference is saved automatically.

User Roles

Different access levels for team management.

  • User: Access all calculation tools
  • Manager: View team calculations and reports
  • Admin: Full system management access
